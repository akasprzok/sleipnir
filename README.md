# Sleipnir

A Loki client.

## Installation

The package can be installed
by adding `sleipnir` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:sleipnir, "~> 0.1.0"}
  ]
end
```

Documentation may be found at <https://hexdocs.pm/sleipnir>.
